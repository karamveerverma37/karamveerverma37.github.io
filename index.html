About Me

I am a postdoctoral researcher in the Uzun Lab at Penn State University, working on single-cell multi-omics data integration, gene regulatory network (GRN) inference, and machine learning applications in aging and precision medicine.

Research Interests

  • Single-cell GRN inference and benchmarking
  • Multi-omics integration (scRNA-seq, scATAC-seq, methylation)
  • Deep learning for genotype-to-phenotype prediction
  • Drug discovery using docking/simulation
  • Cell-type-specific aging clocks
  • Metagenomic data analysis

Tools & Projects

  • Bench-GRN: Benchmarking framework for GRN inference (GitHub)
  • CarcinoHPVPred: Tool for HPV-associated carcinogenicity prediction (re-hosted)
